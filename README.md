# Pong

Classic Pong Game: A recreation of the iconic Pong game where players control paddles to bounce a ball back and forth. Enjoy classic gameplay in a nostalgic 1 or 2-player mode.

## How to Play

- Use the arrow keys to select mode
- Use W S for Player 1
- Use Up Down arrows for Player 2
